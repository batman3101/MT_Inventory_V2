### **풀스택 기술 리스트 (Full Stack List)**

*   **프론트엔드 (Client-side):**
    *   **언어:** TypeScript
    *   **프레임워크:** React.js
    *   **빌드 툴:** Vite
    *   **스타일링:** Tailwind CSS
    *   **UI 라이브러리:** Headless UI / shadcn/ui
    *   **라우팅:** React Router DOM
    *   **상태 관리:** Zustand (또는 Redux Toolkit)
    *   **데이터 페칭:** Axios / TanStack Query

*   **백엔드 (Server-side):**
    *   **언어:** TypeScript
    *   **프레임워크:** Node.js, Express.js
    *   **데이터베이스:** Supabase (PostgreSQL)
    *   **DB 클라이언트:** Supabase.js / node-postgres (pg)
    *   **인증:** JSON Web Tokens (JWT), bcrypt

*   **개발/배포 (DevOps):**
    *   **패키지 매니저:** NPM 또는 Yarn
    *   **테스팅:** Jest, Supertest, React Testing Library
    *   **배포:** Docker, Vercel, Render
