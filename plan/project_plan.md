### **프로젝트 고도화 기획서**

**1. 프로젝트 명:** CNC 재고 관리 시스템 V3

**2. 목표:**
기존 Python/Streamlit 애플리케이션을 최신 웹 기술 스택으로 전환하여, 더욱 향상된 사용자 경험(UX), 빠른 반응성, 높은 확장성 및 유지보수성을 갖춘 웹 애플리케이션을 구축합니다.

**3. 핵심 기능:**
*   **대시보드:** 재고 현황, 최근 입출고 내역 등 핵심 지표를 시각적으로 제공합니다.
*   **재고 관리:** 실시간 재고 조회, 강력한 검색 및 필터링 기능을 제공합니다.
*   **부품 관리 (CRUD):** 부품 정보(사진 포함)의 생성, 조회, 수정, 삭제 기능을 제공합니다.
*   **입출고 관리:** 모든 재고 변동 내역을 기록하고 추적합니다.
*   **공급처 관리:** 공급처 정보의 생성, 조회, 수정, 삭제 기능을 제공합니다.
*   **사용자 및 권한 관리:** 관리자 전용 페이지에서 사용자 생성 및 역할(Admin, Manager, Viewer) 부여 기능을 제공합니다.
*   **인증:** JWT(JSON Web Token) 기반의 안전한 사용자 로그인 및 세션 관리를 구현합니다.

**4. UI/UX:**
*   **React**와 **Tailwind CSS**를 사용하여 전문적이고 현대적인 디자인을 구현합니다.
*   모바일과 데스크톱을 모두 지원하는 반응형 웹 디자인을 적용합니다.
*   직관적인 데이터 입력 폼과 동적 데이터 테이블을 통해 사용 편의성을 극대화합니다.
